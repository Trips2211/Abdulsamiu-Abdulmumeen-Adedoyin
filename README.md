# CraveHub – Campus Food Delivery Platform

CraveHub is a food delivery platform designed for university campuses where students can order food from vendors and runners deliver using bicycles.

## Features

- Vendor dashboard
- Student ordering system
- Cart and checkout
- Runner delivery system
- Wallet and order tracking

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Supabase

Payment:
- Paystack

Hosting:
- Netlify
